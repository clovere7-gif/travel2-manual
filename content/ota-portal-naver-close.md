# 네이버 객실마감, 요금수정

## 1. 객실마감

### 1. 스마트플레이스 로그인

[https://new.smartplace.naver.com/](https://new.smartplace.naver.com/) 접속 — 아이디 동일

![스마트플레이스 로그인 화면](images/ota/naver/ota-portal-naver-reserve-08.PNG)


### 2. 시설 접속

- 변경하여야 하는 시설 접속

![시설선택 팝업 화면](images/ota/naver/ota-portal-naver-reserve-09.PNG)

스마트플레이스 접속 후 ① > 클릭하여 ② 변경하여야 하는 시설 선택


### 3. 기본화면 → 간단예약관리

- 선택 시 기본화면 확인

![간단예약관리 객실관리 화면](images/ota/naver/ota-portal-naver-reserve-10.PNG)

예약 → 간단예약관리 → 객실관리페이지 접속

![간단예약관리 객실관리 화면](images/ota/naver/ota-portal-naver-reserve-10.PNG)


### 4. 객실마감 설정

![객실마감 토글 버튼 화면](images/ota/naver/ota-portal-naver-reserve-11.PNG)

해당 버튼으로 빈공간 클릭 — 초록색은 열림, 회색은 닫힘

## 2. 요금수정

### 1. 객실목록 접속

상동 시설 선택 후 객실목록 클릭

![객실목록 화면](images/ota/naver/ota-portal-naver-reserve-12.PNG)


### 2. 객실 선택

변경하려는 객실 선택

![일정설정 달력 화면](images/ota/naver/ota-portal-naver-reserve-13.PNG)

- 변경하려는 객실 선택 → 기본정보 → 일정설정


### 3. 날짜 선택

![달력 날짜선택 화면](images/ota/naver/ota-portal-naver-reserve-14.PNG)

- 달력모양에서 변경하려는 날짜 선택

![기간 재선택 화면](images/ota/naver/ota-portal-naver-reserve-15.PNG)

- 날짜 선택 후 기간 재선택

### 4. 금액 입력

![요일별 금액입력 화면](images/ota/naver/ota-portal-naver-reserve-16.PNG)

- ⓐ 선택된 ⓑ 기간에 영업유무 및 객실수, 판매가 기준으로 금액 입력

> ※ 특수일(공휴일 등)은 요금표 하단에 따로 생성됨 — 추가로 입력 필요 하단

![공휴일 설정 화면](images/ota/naver/ota-portal-naver-reserve-17.PNG)
