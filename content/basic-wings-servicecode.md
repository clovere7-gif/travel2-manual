# 서비스코드 등록

## 윙스 - 서비스코드 등록

고마스터 - 랜딩구매 / 부대시설이용문자에서 고객구매 > 고마스터에 자동등록

![이미지](images/wing/basic-wings-servicecode-01.PNG)

고마스터 - 패키지상품구매 / OTA상품 구매시 객실과 함께 인입, 예약건 생성시 패키지상품으로 등록

![이미지](images/wing/basic-wings-servicecode-02.PNG)

1 예약건 진행후 > 일별요금 > 스페셜서비스 > 추가

![이미지](images/wing/basic-wings-servicecode-03.PNG)

2 추가 클릭시 > 서비스코드 등록 생성 > 날짜 서비스 > 수량 > 금액 입력

![이미지](images/wing/basic-wings-servicecode-04.PNG)

2-1 일자 / 날짜 선택 
2-2 서비스 / 스페셜서비스 중 서비스코드 클릭

![이미지](images/wing/basic-wings-servicecode-05.PNG) ![이미지](images/wing/basic-wings-servicecode-05.PNG)

2-3 수량, 금액 확인 / 예약에 맞는 수량입력 > 금액(자동설정) 금액 다를시 확인후 수정

![이미지](images/wing/basic-wings-servicecode-06.PNG)
