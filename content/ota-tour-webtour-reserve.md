# 예약

## 1. 메일 확인

담당메일 로그인 후 개인메일함 **여행사**에서 예약 확인

![하이웍스 메일함 여행사 탭](images/ota/tour/ota-tour-webtour-reserve-01.PNG)

## 2. 예약진행 (연동)

예약 알림메일 확인 후 알림메일에서 예약

![웹투어 알림메일 바우처](images/ota/tour/ota-tour-webtour-reserve-02.PNG)

**알림메일(바우처) 확인 항목**

1. 예약번호
2. 날짜
3. 고객정보
4. 객실 및 상품정보
5. 입금가 / 수수료 15%

> 예약완료 후 관리자 사이트에서 예약확정 처리

![관리자 사이트 로그인 화면](images/ota/tour/ota-tour-webtour-reserve-03.PNG)

**예약확정 처리 순서**

1. 시설 선택
2. 연동건은 윙스번호 기재되어 있음 / 미연동은 오션투유의 경우 성함+입실(OOO) 기재
3. 예약완료 시 상태 확인
4. 상태 → 예약완료로 선택한 후에 저장 클릭

![예약확정 처리 화면](images/ota/tour/ota-tour-webtour-reserve-04.PNG)
