# 해외 OTA 공통 및 특이사항

## 1. 예약 진행

- A. 대량등록 — 아고다, 익피만 진행
- B. 메일 읽는 순서 → 취소 or 답변 메일 선진행 → 예약메일 진행

## 2. 채널별 특이사항

### 아고다

**대량등록 주의사항**
① 오전에 대량등록은 하되, 추후부터는 예약건 수기로 찍기
② 엑셀 없다면 선임자가 오전에 대량등록 1회 진행 — 예약은 예약담당자가
③ **중요** — 수정건은 대량등록이 안되기 때문에 대량등록할때 유의하기
④ **중요** — 오션투유 대량등록시 매칭안되는 내용이 많으니 확인 후 객실명 및 수정사항 변경하기

**그 외**
- 채널 혹은 고객 답변 메일은 누락없이 답변 나가기 **필수!**
- 키오스크 연락방법 → 번호 있으면 문자 / 가능하면 메일 / 없으면 아고다 채팅

### 익스피디아
- 대량등록 후 입금가 수정은 검색해서 하라고 교육하기
- 동탄, 역삼은 메일주소도 메모에 남겨두기

### 부킹닷컴
- 객실 여러 개 예약일시 → 윙스 예약건 여러 개 생성됨

### 웹베즈
- 윙스 판매가 나눠서 진행 — 예약번호는 각 예약건의 예약번호 전부 기입
- OTA 예약번호로 윙스 검색 안되기 때문에 이름으로 검색 / 취소건은 CXL로 인입
- 동탄 — 예약 메일 바우처에 회신으로 키오스크 문자 보내주기
- 키오스크 연락 방법 → 채팅 → 메일 → 메시지

## 3. 판매가 등록시 특이채널

| 채널 | 계산방법 | 비고 |
|------|----------|------|
| 오마호 | 입금가 ÷ 0.85 (입금가/15%) | 예약생성시 버튼 있음 |
| 호텔패스 | 입금가 ÷ 0.85 (입금가/15%) | 예약생성시 버튼 있음 |
| 웹베즈 | 입금가 ÷ 0.8 | 수기로 계산 |

## 4. 관리번호 기입 채널

- **씨트립** — 오션투유만 확정처리 진행 / ★ 무취건도 취소확정처리
- **호텔패스**

## 5. 요금수정 특이사항

| 채널 | 방법 |
|------|------|
| 아고다 | 인원수별 요금 및 숙박규정을 통해서 요금 변경 |
| 익스피디아 | 특이사항 없음 |
| 씨트립 | 요금수정 하고자 하는 일자만 선택 |
| 부킹닷컴 | 요금수정 하고자 하는 일자만 선택 |
| 오마호 | 레이트 클릭 → 요금수정 → gross(판매가)에 넣고 contract 0원으로 수정 |
| 트립비토즈 | 미판매(노란색)는 우클릭 후 판매로 돌려서 수정 |

**웹베즈** (상세)
- 요금수정 메일발송을 통해 진행
- 조식 바베큐 제거 후 엑셀에 번역 기능 사용해서 메일 보내기
- 객실마감 → 제일 우측 제일 하단 → 객실 판매 마감
  - ㄴ 업데이트 누른 후 저장까지 눌러야 함

## 6.요금수정 및 컨펌

- 오션투유 일반 테라스 — 11~3월 / 테라스 워터스파 — 4~10월
- 컨펌 — 각 관리자페이지 전부 들어가서 확인 후 마지막에 네이버 메타채널에 변경일자 금액 체크

> **채널별 체크 (채널파악, 문제점, 횟수 등 데이터 필요)**
1. 문제 많이 생기는 사이트 혹은 오류 나는 사이트는 요금 변경 후 문제없는지 메일로 확인 요청 / 계속 귀찮게 하고 책임소재 확실하게 함
  - ㄴ 긴(중요한) 기간 요금수정 시 문의 메일 작성 — 중요 일자 판단 기준은 선임자가 결정
2. 리컨펌 선임도 하고 신입직원도 시킴 — 잘못된 경우 경고 후 조치
3. 판매페이지에서도 최저가로 확인함

> 엑셀 요금표 공백 없애는 방법도 같이 알려주기
