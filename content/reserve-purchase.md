# 리저브클럽 구매

## 프로세스

카톡 발송 → 리저브 구매 → 리저브 이용


## 구매 경로

| 구분 | 내용 |
|------|------|
| 카카오톡 광고 | 수신된 카카오톡 광고 메시지 클릭 (구매링크 연결) |
| 공식 홈페이지 | 공식 홈페이지 접속 → 리저브클럽 메뉴 클릭 → 리저브클럽 구매하기 |
| 네이버 스마트스토어 | 인입량 적음 (리저브 담당자가 확인) |

**구매링크**

| 시설 | 링크 |
|------|------|
| 오션투유 | https://ocean2you.co.kr/gift |
| 아이파크 | https://iparksuite-sokcho.com/gift |
| 마리비스타 | https://marivista.co.kr/reserve-club/special-price |


## 결제 수단 및 유의사항

| 결제 수단 | 유의사항 |
|------|------|
| 카드 결제 (KCP) | - |
| 카카오페이 | 99상품 구매 시 사용 불가 (마리비스타는 가능) |
| 네이버페이 | 마리비스타는 불가 |

> ※ 무이자 할부 불가 — 리저브 상품 결제 시 모든 카드의 무이자 할부 혜택 적용 불가 (일반 할부만 가능)

## 구매 시 고객 안내 카톡 (자동발송)

![구매정보 카톡 / 구매안내 카톡](images/reserve/reserve-03.PNG)

> 고객이 인증번호를 분실했거나 알림톡을 받지 못한 경우 관리자 페이지에서 재발송 가능
> 경로 — 리저브 관리자페이지 → 사용자 관리 → 회원
> ① [구매] 버튼 클릭 — 인증정보(고객인증번호 등) 발송
> ② [안내] 버튼 클릭 — 리저브클럽 주요 이용 안내문 발송


## 구매 후 고객정보 확인 경로

### 고마스터 구매리스트

경로 — 고마스터 → 예약관리 → 리저브클럽 구매리스트

![고마스터 화면](images/reserve/reserve-04.PNG)

| 가능 | 불가 |
|------|------|
| 고객 연락처, 성함 등 | 고유번호, 유효기간 확인 |
| 결제 금액, 수단 등 | 정보 수정 |


### 리저브클럽 관리자페이지

경로 — 리저브클럽 관리자 → 사용자관리 → 회원

![관리자페이지 화면](images/reserve/reserve-05.PNG)

| 가능 | 불가 |
|------|------|
| 회원 정보 | 결제수단 확인 |
| 이름, 연락처, 유효기간 등 | - |
| 조회 및 수정 | - |
| 이용권 상태 관리 | 고객 고유번호 수정 |
| 일련번호 확인 | - |
| 연동오류 고객수기등록 | - |
| 구매/안내 알림톡재발송 | - |
| 예약 인벤토리 관리, 요금 설정 | - |
