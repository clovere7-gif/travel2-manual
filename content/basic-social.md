# 사회생활 기본

## 1. 기초 용어 - 시간

어제 = 전일, 작일 / 저번주 = 전주, 작주 / 저번달 = 전월, 작월
'전' 은 이전이라는 의미 / '작' 은 직전이라는 의미
한자어 사용시 전일,전주,전월 을 많이 사용함

오늘 = 금일, 당일 / 이번주 = 금주, 당주 / 이번달 = 금월, 당월
'금' 은 현재라는 의미 / '당' 은 시점을 기준으로 현재 의미
한자어 사용시 금일,당일,금주,당월 을 많이 사용함

내일 = 명일, 익일 / 다음주 = 차주, 내주 / 다음달 = 익월, 차월
'차' 는 한자어로 다음이라는 의미 / '명' 은 밝아오다 의미(다음이 밝아온다)
'래(내)' 는 다가온다는 의미 / '익' 시점을 기준으로 기간을 의미
한자어 사용시 명일,익일,차주,익월 을 많이 사용함

그외 격일 - 하루걸러

## 2. 기초 용어 - 업무

결재 - 부하 직원이 제출한 문서를 상관이 검토 승인하는것
결제 - 돈을 결제하다 결재와 다름
기안 - 결재가 필요한 문서를 작성하여 결재권자에게 승인을 요청하는 행위
반려 - 승인을 요청한 문서를 처리하지 않고 되돌려보냄
근태 - 출근 상태 및 근무 태도
지양 - 하지않음
지향 - 함

수기 - 글을 자기 손으로 직접 씀. 일일이 직접 입력해야 하는 것
참고 - 자료나 정보를 활용하거나 도움이 되도록 하는 것을 의미
참조ⓐ - 자료나 정보를 다른 대상과 비교하거나 관련시켜 본다는 의미
참조ⓑ - 메일이나 회신시 관련자를 추가함 영어 CC
송부 - 편지나 물품을 보냄
별첨 - 서류를 따로 첨부함

하기 - 본문에 내용을 위에서 아래방향으로 가리킬 때
상기 - 본문에 내용을 밑에서 위방향으로 가리킬 때
상동 ①서로 같음 ②위에 적힌 사실과 같음
상반기 - 어느 기간을 둘로 나눈 그 앞쪽의 반 / 보통 1년중 1~6월을 뜻함
하반기 - 어느 기간을 둘로 나눈 뒤쪽의 반 / 보통 1년중 7~12월을 뜻함
반기 - 어느 기간을 둘로 나눈것 / 한해의 절반 6개월을 뜻함 예) 상반기 하반기
분기 - 1년을 3개월씩 4개로 나눈것 / 1분기(1~3월), 2분기(4~6월), 3분기(7~9월), 4분기(10~12월)
