# 사회생활 기본 용어

## 시간 관련 용어

| 용어 | 의미 |
|------|------|
| 어제 / 전일 / 작일 | 하루 전날 |
| 자번주 / 전주 / 작주 | 지난 주 |
| 자번달 / 전월 / 작월 | 지난 달 |
| 오늘 / 금일 / 당일 | 현재 날짜 |
| 이번주 / 금주 / 당주 | 현재 주 |
| 이번달 / 금월 / 당월 | 현재 달 |
| 내일 / 명일 / 익일 | 다음 날 |
| 다음주 / 차주 / 내주 | 다음 주 |
| 다음달 / 익월 / 차월 | 다음 달 |
| 격일 | 하루 건너 |

## 업무 관련 용어

| 용어 | 의미 |
|------|------|
| 결재 | 부하 직원이 제출한 문서를 상관이 검토 승인하는 것 |
| 결제 | 돈을 결제하는 것 (결재와 다름) |
| 기안 | 결재가 필요한 문서를 작성해 결재권자에게 승인 요청 |
| 반려 | 승인을 요청한 문서를 처리하지 않고 되돌려보냄 |
| 근태 | 출근 상태 및 근무 태도 |
| 수기 | 글을 자기 손으로 직접 씀 |

> ⚠️ 결재와 결제는 발음이 같지만 의미가 다릅니다. 보고서 작성 시 헷갈리지 않도록 주의하세요.
