# 아이파크

## 기본정보

| 구분 | 내용 |
|------|------|
| 부대시설 | 조식당, 세미나실, 룸서비스, 카페테리아, 키즈존 |
| 펫룸 | 더블, 트윈, 더블트윈, 온돌, 스위트 가능 (코드 있음) |
| 주차 | 가능 (약간 부족) |
| 특이사항 | - |
| 비고 | 취사불가 |


## 객실코드

| 코드 | 상품명 |
|------|------|
| SOD | 시티뷰 스탠다드, 슈페리어 더블 |
| SPD | 파셜/파노디럭스 더블 |
| SPT | 시티뷰 스탠다드, 슈페리어 트윈 |
| SCD | 베리어 프리 (장애인 객실) |
| DFT | 파셜/파노디럭스 트윈 |
| DFD | 시티/파셜/파노디럭스 더블트윈 |
| DOD | 시티/파셜/파노디럭스 온돌 |
| PFT | 프리미어 디럭스 더블트윈 |
| OCS | 파노라마 오션뷰 스위트 |
| OCF | 파노라마 오션뷰 그랜드 스위트 |
| PDB | 펫룸 파셜 디럭스 더블 |
| PTW | 펫룸 파셜 디럭스 트윈 |
| PDT | 펫룸 파셜 디럭스 더블트윈 |
| PSU | 펫룸 파셜 디럭스 스위트 |
| PHA | 펜트하우스 패밀리 |
| PHB | 펜트하우스 주니어 |
| PHC | 펜트하우스 스위트 |
| PHD | 펜트하우스 스위트 |
| PHE | 펜트하우스 스위트 |
| PHF | 펜트하우스 스위트 |
| PHG | 펜트하우스 프리미어 |
| PHH | 펜트하우스 프리미어 |
| PHI | 펜트하우스 그랜드 |


## 업그레이드 순서

시티뷰 → 파셜 → 파노라마

> ※ 객실별 구조가 같음
> ※ 트윈 → 더블트윈 → 프리미어 더블트윈 가능
> ※ 스위트급은 웬만하면 업그레이드 지양
