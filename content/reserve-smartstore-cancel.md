# 스마트스토어 취소


## 1. 신규주문(발주 후) 취소

1. 신규주문(발주 후) 상태일 때 고객이 취소요청 가능 → 취소관리에 취소요청으로 인입
2. 신규주문(발주 전) 상태일 때만 고객 취소 접수 시 자동취소 가능 — 배송없음이기 때문에 해당 없음
3. 거부 시 취소접수 불가 / 고객센터 이용해야 함

> ※ 신규주문(발주 후) 상태는 아직 고유번호 생성 전이라 확인 없이 취소 가능

![신규주문 취소 화면](images/reserve/reserve-14.PNG)

## 2. 배송완료 취소

1. 배송완료 상태일 때 고객이 취소요청 가능 → 반품관리에 반품요청으로 인입
2. 거부 시 취소접수 불가 / 고객센터 이용해야 함

> ※ 배송완료 상태는 고유번호 생성 후라 객실 사용여부 확인 후 취소 가능

![배송완료 취소 화면](images/reserve/reserve-15.PNG)

## 3. 구매확정 취소

1. 구매확정 시 취소 불가 / 고객이 고객센터로 취소 접수해야 함
2. 고객센터 인입 시 구매확정 내역 → 구매확정 후 취소처리로 취소

> ※ 구매확정 상태는 고유번호 생성 후, 발송 후 2주가 지난 시점일 수 있어 객실 사용여부 및 발송일자 확인 후 취소 가능
> ※ 구매확정은 배송완료 후 8일 자동 구매확정 또는 고객 직접 구매확정
