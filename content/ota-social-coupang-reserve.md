# 쿠팡 예약

## 업무범위

1. 메일 확인
2. 예약진행 (연동)
3. 객실마감, 요금수정
4. 완료예약확인

> ※ 쿠팡 관리자 사이트는 로그인이 자주 풀리는 오류가 있습니다. 재시도하면 됩니다.


## 예약

### 1. 메일 확인

담당메일 로그인 후 개인메일함 **소셜**에서 예약 확인

![하이웍스 소셜 메일함](images/ota/coupang/ota-social-coupang-reserve-01.PNG)


### 2. 예약진행 (연동)

예약 알림메일 확인 후 하단 **예약확인하러 바로 가기** 클릭

![쿠팡 알림메일 바우처](images/ota/coupang/ota-social-coupang-reserve-02.PNG)

![관리자 사이트 예약내역](images/ota/coupang/ota-social-coupang-reserve-03.PNG)

**관리자 사이트 확인 항목**

1. 고객정보
2. 예약번호
3. 객실 및 상품정보
4. 옵션
5. 날짜
6. 판매가, 입금가
7. 대리 구매한 경우 구매자와 투숙자 이름을 같이 기재하여 혼동 없게 할 것

![예약/취소 관리 목록](images/ota/coupang/ota-social-coupang-reserve-04.PNG)

> 예약 완료 후 상단 **목록으로 돌아가기** 클릭
> 예약완료 후 관리번호등록(초기예약 시 공란)에 오션투유 성함+입실(OOO) 혹은 윙스 번호 입력 후 저장


## 객실마감, 요금수정

### 객실마감

![요금/수량 전체 상품 보기](images/ota/coupang/ota-social-coupang-reserve-05.PNG)

![달력 및 객실 마감 설정](images/ota/coupang/ota-social-coupang-reserve-06.PNG)

**관리자페이지 순서**

1. 요금/수량 선택
2. 전체 상품 보기 클릭
3. 등록상품 전체보기 팝업 → 변경하려는 시설 검색 → 시설 선택
4. 변경하려는 시설 달력에서 일자 선택
5. 변경하려는 객실 선택
6. 판매상태 클릭 변경


### 요금수정

![요금수정 화면](images/ota/coupang/ota-social-coupang-reserve-07.PNG)

1. 요금/수량 선택
2. 달력에서 변경하려는 일자 선택
3. 변경하려는 ①객실 선택
4. ②재고수량 변경 필요 시 변경
5. ③판매가 기준으로 변경
