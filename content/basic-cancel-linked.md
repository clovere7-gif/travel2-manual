# 예약 취소 - A.연동취소 B.미연동 취소

> ★ 관리하는 자사, 여기어때(모텔), 온다, 네이버 미연동 나머지 채널 연동

## A. 연동 예약건 취소 (당일앱, 해외, 소셜, 여행사 동일)

> ★ 미연동(오션투유 제외)의 경우 윙스 수기 취소 및 고마스터 취소 / 오션투유 윙스 없음 고마스터만 취소

### 해외 채널 (아고다 등)

- 확정/변경/취소 메일 인입 경로: 해외 예약 담당 메일
- 해당 메일로 취소 바우처 인입 시 → **취소 수수료 확인** → 0원으로 되어있다면 하단 절차 진행
- 고마스터 OTA예약번호 검색 → 메모에 취소 수수료 기재 → 주문 취소 버튼 클릭
- CMS 연동 건의 경우, 산하에서 윙스 자동 취소 처리되며 고마스터만 취소하면 됨

### 당일앱 (야놀자, 여기어때 호텔)

- 확정/변경/취소 메일 인입 경로: 야놀자/여기어때 담당 메일
- 해당 메일로 취소 바우처 인입 시 → **취소 수수료 확인** → 0원으로 되어있다면 하단 절차 진행
- 고마스터 OTA예약번호 검색 → 메모에 취소 수수료 기재 → 주문 취소 버튼 클릭
- CMS 연동 건의 경우, 산하에서 윙스 자동 취소 처리되며 고마스터만 취소하면 됨
