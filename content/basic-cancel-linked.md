# 예약 취소 - A.미연동 취소 B.연동 취소

## A. 미연동 예약건 취소 (당일앱, 소셜, 여행사, 자사, 해외 동일)

당일앱 - 여기어때모텔/선착순특가, 자사, 네이버전체, 온다전체, 여행사B2B, 해외 오션(고성계정)

> ★ 미연동의 경우 윙스 수기 취소 및 고마스터 취소 / 누락주의

## B. 연동 예약건 취소 (당일앱, 해외, 소셜, 여행사, 해외 동일)

### 당일앱 (야놀자, 여기어때 호텔)

![이미지](images/cancel/basic-cancel-linked-01.PNG)

- 확정/변경/취소 메일 인입 경로: 야놀자/여기어때 담당 메일
- 해당 메일로 취소 바우처 인입 시 → **취소 수수료 확인** → 0원으로 되어있다면 하단 절차 진행
  
![이미지](images/cancel/basic-cancel-linked-02.PNG)

- 고마스터 OTA예약번호 검색 → 메모에 취소 수수료 기재 → 주문 취소 버튼 클릭
- CMS 연동 건의 경우, 산하에서 윙스 자동 취소 처리되며 고마스터만 취소하면 됨

![이미지](images/cancel/basic-cancel-linked-03.PNG)

- 해당 메일로 취소 바우처 인입 시 → **취소 수수료 확인** → 0원으로 되어있다면 하단 절차 진행

![이미지](images/cancel/basic-cancel-linked-04.PNG)

- CMS 연동 건의 경우, 산하에서 윙스 자동 취소 처리되며 고마스터만 취소하면 됨
