**3-1.**
- 취소대기 — 현재는 없음 / 취대건 있을경우 자사 아니여도 확인 후 웬만하면 처리
- 예약변경 — 부대시설건 많음
  - 날짜 지났을 경우 전화번호 없애고 확정, 예약변경해제
  - 그 외 건 자사 아니여도 확인 후 웬만하면 처리

**3-2.**
- 패키지 확인 후 서핑 있을 경우 → 고마스터 생성 후 서핑업체에 카톡 발송
- 입장권 서핑 있을 경우 → 서핑업체에 카톡 발송 / 전일 것까지 확인하면 더 좋습니다
