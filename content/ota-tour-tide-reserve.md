# 타이드스퀘어(투어비스) 예약

## 업무범위

1. 메일 확인
2. 예약진행 (연동)
3. 객실마감, 요금수정
4. 완료예약확인


## 예약

### 1. 메일 확인

담당메일 로그인 후 개인메일함 **여행사**에서 예약 확인

![하이웍스 메일함 여행사 탭](images/ota/tour/ota-tour-tide-reserve-01.PNG)


### 2. 예약진행 (연동)

예약 알림메일 확인 후 알림메일에서 예약

![타이드스퀘어 알림메일 바우처](images/ota/tour/ota-tour-tide-reserve-02.PNG)

**알림메일(바우처) 확인 항목**

1. 예약번호
2. 객실 및 상품정보
3. 날짜
4. 고객정보
5. 입금가 / 수수료 15%
6. 윙스 번호


## 객실마감, 요금수정

### 객실마감

![블럭설정 화면](images/ota/tour/ota-tour-tide-reserve-03.PNG)

![달력 날짜선택 + 블럭수정 팝업](images/ota/tour/ota-tour-tide-reserve-04.PNG)

![전체선택 수정 완료 화면](images/ota/tour/ota-tour-tide-reserve-05.PNG)

1. 관리자 사이트 요금&블럭 → 블럭설정 클릭
2. 마감해야 하는 객실 선택
3. 마감일자 기간 확인
4. 조회

날짜 선택 시 객실변경 칸 생성

전체선택 혹은 마감해야 하는 객실 선택 후 수정 클릭 완료


### 요금수정

![요금등록 화면](images/ota/tour/ota-tour-tide-reserve-06.PNG)

![달력 + 수정 팝업 화면](images/ota/tour/ota-tour-tide-reserve-07.PNG)

1. 관리자 사이트 요금&블럭 → 요금등록 클릭
2. 룸온리 → 변경해야 하는 객실 선택 → 조회
3. 날짜 선택 시 객실변경 칸 생성 → 판매가 기준으로 입력
4. 수정
5. 저장
