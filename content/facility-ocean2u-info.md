# 오션투유

## 기본정보

| 구분 | 내용 |
|------|------|
| 부대시설 | 조식당, 바베큐장, 세미나실, 룸서비스, 기타 - 해수욕장(매우 가까움) |
| 펫룸 | 로얄스위트 제외 전타입 가능 (코드 없음) |
| 주차 | 가능 |
| 특이사항 | - |
| 비고 | 전객실 가벼운 취사가능 |


## 객실코드

| 코드 | 상품명 |
|------|------|
| SFS | 스탠다드 / 슈페리어호텔 / 디럭스 포레스트 |
| SOS | 사이드 오션뷰 |
| POS | 파노라마 오션뷰 |
| SOT | 사이드 오션뷰 테라스 |
| POT | 파노라마 오션뷰 테라스 |
| ROT | 로얄 스위트 |


## 업그레이드 순서

스탠다드 → 슈페리어 → 디럭스 → 포레스트 → 사이드 오션뷰 → 파노라마 오션뷰

> ※ 포레스트 하위객실부터 오션뷰객실로 업그레이드 — 테라스까지는 잘 업그레이드하지 않음
> ※ 로얄스위트 업그레이드 없음
