# 위약취소

> ★ 해외는 대부분 입실일 기준 5일 전 무료취소, 이 외 전부 전액 위약금

## 야놀자

- 엑스트라넷: https://partner.yanolja.com/reservation/search

![이미지](images/cancel/basic-cancel-linked-05.PNG)

- 담당 메일로 취소 메일 인입 시 → 고객 취소 수수료 확인
- 고객 취소 수수료 0원이 아닐 경우 하기 절차 진행

> 야놀자 통합계정으로 엑스트라넷 로그인 후 취소 수수료 확인

![이미지](images/cancel/basic-cancel-linked-06.PNG)

- 로그인 후 왼쪽 상단 호텔 클릭 > 예약번호 검색 (화면에서 Ctrl + F 누르고 예약번호 복붙해서 검색)
- 상세 보기 누르면 하기 사진 참고

![이미지](images/cancel/basic-cancel-linked-07.PNG)

- 고객 취소 수수료(전체) = 위약취소 시설 판매가
- 고객 취소 수수료(제휴점 수취분) = 위약취소 시설 입금가

> ※ 상기 내용으로 고마스터 위약취소 시설로 취소 처리

## 여기어때

- 엑스트라넷: https://partner.goodchoice.kr/login

![이미지](images/cancel/basic-cancel-linked-08.PNG)

- 담당 메일로 취소 메일 인입 시 → 고객 취소 수수료 확인
- 고객 취소 수수료 0원이 아닐 경우 하기 절차 진행

- 엑스트라넷 로그인 후 취소 수수료 확인

![이미지](images/cancel/basic-cancel-linked-09.PNG)

- 로그인 후 호텔 선택 → 오른쪽 상단 예약번호 검색

![이미지](images/cancel/basic-cancel-linked-10.PNG)

- 예약건 클릭 > 스크롤하여 취소/환불 정보에서 "취소 상품 차감 금액" = 위약취소 시설 판매가
- 취소 메일에 기재되어 있는 "총 정산 예정금액" = 위약취소 시설 입금가

> ※ 상기 내용으로 고마스터 위약취소 시설로 취소 처리

### 위약취소 금액 계산

| 구분 | 내용 |
|------|------|
| 취소 상품 차감 금액 | 위약취소 시설 판매가 |
| 총 정산 예정금액 (취소 메일 기재) | 위약취소 시설 입금가 |
| 고객 취소 수수료(전체) | 위약취소 시설 판매가 |
| 고객 취소 수수료(제휴점 수취분) | 위약취소 시설 입금가 |

> 상기 내용으로 고마스터 위약취소 시설로 취소 처리
