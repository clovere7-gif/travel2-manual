# 마이리얼트립 예약

## 업무범위

1. 메일 확인
2. 예약진행 (연동)
3. 객실마감, 요금수정
4. 완료예약확인


## 예약

### 1. 메일 확인

담당메일 로그인 후 받은메일함에서 예약 확인

![하이웍스 메일함](images/ota/tour/ota-tour-myrealtrip-reserve-01.PNG)


### 2. 예약진행 (연동)

예약 알림메일 확인 후 알림메일에서 예약

![마이리얼트립 알림메일 바우처](images/ota/tour/ota-tour-myrealtrip-reserve-02.PNG)

**알림메일(바우처) 확인 항목**

1. 예약번호
2. 고객정보 — 영문명으로 들어올 때 많음
3. 객실 및 상품정보
4. 날짜
5. 판매가 / 입금가

예약완료 후 관리자 사이트에서 확정번호 메모 추가

![관리자 사이트 예약목록 + 호텔예약번호 팝업](images/ota/tour/ota-tour-myrealtrip-reserve-03.PNG)

호텔 예약 번호 클릭 → 팝업 → 윙스 혹은 성함+입실 기재


## 객실마감, 요금수정

![요금/재고관리 시설목록 화면](images/ota/tour/ota-tour-myrealtrip-reserve-04.PNG)

![객실 판매상태/재고/금액 변경 화면](images/ota/tour/ota-tour-myrealtrip-reserve-05.PNG)

요금/재고관리 → 호텔/리조트/펜션 → 변경하려는 시설 선택

객실 클릭 시 → 내용 펼쳐짐 → 판매상태 재고 금액 변경
