# 리뷰 - 공통 및 특이사항

## 리뷰 구매 시 참고 및 주의사항

> 🚨 가장 중요: 하드블럭/컴프객실/해외현장지불(후지불, 나중결제)/야놀자(모텔) 9만원 미만 객실은 무조건 구매하면 안 됩니다.
> 구매한 경우 '심수연대리'에게 선메신저 후 무조건 취소해야 합니다.

> 🚨 구글시트 구매 리스트와 상이하게 구매한 경우, 무조건 '심수연대리'에게 메신저로 알립니다.

> 🚨 구매 후 매일매일 금액을 맞춰보는 작업이 필요하며, 월초에 법인카드 카드결제금액 정산작업을 진행하므로 구글시트의 결제금액(신용카드결제금액)은 절대 잘못 입력하면 안 됩니다.

## 채널별 주의사항

| 채널 | 주의사항 |
|------|----------|
| 야놀자(호텔) | 하드블럭상품(마이룸, 쿠폰룸, 초강력특가, 강력특가 기재된 상품) 구매 금지. 구매 시 할인쿠폰 자동적용건은 반드시 해제 |
| 여기어때(호텔) | 컴프상품(선착순특가 기재된 상품) 구매 금지. 구매 시 할인쿠폰 자동적용건은 반드시 해제 |
| 여기어때(모텔) | 페이백상품 구매 금지. 구매 시 할인쿠폰 자동적용건은 반드시 해제 |
| 해외OTA(아고다/익스피디아/씨트립 등) | 자동결제예정일(카드결제 승인이 나중에 결제되는 건), 현장지불, 후지불, 나중에결제 등은 구매 금지. 반드시 온라인결제, 지금결제 상품만 구매 |
| 역삼 | 대실상품(DAY USE) 구매 금지 |
| 야놀자(모텔) | 9만원 이상 상품으로만 구매. 9만원 미만 상품은 첫구매 시 컴프로 들어와서 구매하면 안 됨 |

## 하드블럭 객실타입 정리

> ⚠️ 아래 기재된 객실타입은 어느 채널에서든 구매하면 안 됩니다. 구매 가능한 객실타입은 구글시트 상단에 별도로 기재되어 있으니 참고하세요.

| 시설 | 구매 금지 객실타입 |
|------|----------|
| 아이파크 | 시티뷰 디럭스 더블 / 파셜디럭스 더블 |
| 마리비스타 | 시티뷰 스탠다드 더블 / 시티뷰 스탠다드 트윈 |
| 오션투유 | 스탠다드 포레스트(4인실) / 슈페리어 포레스트(4인실) |
| 역삼아르누보 | 슈페리어 더블(취사불가) |
