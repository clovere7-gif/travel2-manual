# 아르누보

## 기본정보

| 구분 | 내용 |
|------|------|
| 부대시설 | 없음 |
| 펫룸 | 더블룸 가능 (코드 없음) |
| 주차 | 불가 |
| 특이사항 | 15평 트윈 없음 — 19평 트윈으로 트윈&패밀리 배정, 노출은 트윈~패밀리까지 진행 |
| 비고 | 호텔=취사불가 / 레지던스=취사가능 |


## 객실코드

| 코드 | 상품명 |
|------|------|
| SPD | 스탠다드더블호텔 / 슈페리어호텔 / 슈페리어레지던스 |
| DDB | 디럭스더블 |
| DTW | 스탠다드트윈호텔 / 슈페리어트윈호텔 / 슈페리어트윈레지던스 / 디럭스트윈 |
| DFT | 디럭스 패밀리 |
| SUT | 스위트 |


## 업그레이드 순서

스탠다드 → 슈페리어 → 디럭스

> ※ 패밀리 업그레이드 없음 — 트윈과 패밀리는 요세트 1개 차이로 상호호환 가능
> ※ 스위트 업그레이드 없음
