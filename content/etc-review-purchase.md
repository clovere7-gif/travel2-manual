# 리뷰 - 구매

## 리뷰구매 순서

> ⚠️ 구매리스트가 있는 경우, 구매는 무조건 17시 이전까지 전직원이 진행해야 합니다.

1. 매일매일 구글시트 > 리뷰구매2025-08이후 시트에서 구매리스트를 확인합니다.
2. 구글시트에 기재된 구매건이 있는 경우, 구매일자에 맞춰서 투숙일자/시설명/판매채널을 확인 후 구매합니다.
   - 예) 구매일 5/7, 투숙일 5/7인 경우 → 5월 7일 체크인건으로 구매
   - 예) 구매일 5/7, 투숙일 5/9인 경우 → 5월 9일 체크인건으로 구매

> ⚠️ 리스트에 기재된 투숙일자/시설명/판매채널과 다르게 구매한 경우, 취소하지 말고 일단 심수연대리에게 메신저로 알립니다.

3. 구글시트 결제금액/입금가/구매 란을 수기로 작성합니다.
   - **결제금액**: 신용카드 결제금액으로 입력 (채널의 판매가는 무시하고, 무조건 신용카드로 결제된 금액 기재)
   - **입금가**: 채널의 입금가로 입력
   - **구매**: 구매한 경우 O 표시

4. 해당 채널의 예약건을 찾아서 고마스터 관리자추천을 등록합니다.

> ⚠️ 구매한 채널과 인입된 채널이 상반될 경우, 인입된 채널로 관리자추천을 찍고 구글시트 비고란에 내용을 기재합니다.
> 예) 아고다 채널 구매, 익스피디아 채널 인입 시 → 고마스터: 익스피디아로 찍기 / 구글시트 비고란: "아고다 구매, 익스피디아 인" 기재
>
> 고마스터 판매가 = 신용카드 결제금액, 고마스터 입금가 = 채널의 입금가, 관리자추천 등록 시 리뷰 버튼 클릭 후 찍기

5. 구매한 채널 및 시설이 CMS와 연동된 경우, 윙스에 들어가서 본인 구매건을 윙스에서 수기로 취소 처리합니다.
6. 매일매일 구글시트 > 후기란에 기재된 후기 작성일자에 맞춰서 리뷰구매건의 리뷰 후기를 남깁니다. 남긴 후 후기란에 O 표시합니다.
