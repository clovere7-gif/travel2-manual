# 리저브클럽 이용

## 프로세스

카톡 발송 → 리저브 구매 → **리저브 이용**

## 고객 로그인

고객은 별도의 회원가입 없이 고객정보(고객인증번호/성함/연락처)를 입력하여 이용 가능

경로 : 홈페이지 → 리저브클럽 → 마이페이지

정보 확인: 최초 구매 시 카카오톡으로 발송된 ‘구매 알림톡’ 내에서 인증번호와 예약자 정보를 확인 가능

### 고객 마이페이지 주요 기능

| 구분 | 내용 |
|------|------|
| 정보 조회 | 이름, 연락처, 고객인증번호, 상품명, 유효기간 확인 |
| 정보 수정 | 연락처만 수정 가능 (이름, 고객인증번호, 유효기간 수정 불가) |
| 예약 현황 | 전체 예약 현황, 보유 및 사용 완료된 이용권 현황 확인 |
| 예약 취소 | 입실 2일 전까지 고객 직접 취소 가능 |
| 예약 가능 여부 | 날짜별 객실 마감일 및 예약 가능 일자 실시간 확인 가능 |
| 추가 요금 확인 | 객실 업그레이드(펫룸 등) 및 특정 일자 선택 시 발생하는 추가 요금 확인 |
| 부대 시설 사용 | 현장에서 마이페이지를 통한 현장 사용하기 기능 사용 가능 |

## 예약

### 고객 예약

경로 : 홈페이지 → 리저브클럽 메뉴 → 마이페이지 로그인 → 날짜 선택 및 객실 현황 확인
진행: 날짜 선택 및 객실 현황 확인 (객실 마감 여부 및 추가 요금 즉시 확인 가능)
옵션: 객실 타입 변경, 일자 조정, 펫룸 선택 등 업그레이드

### 예약 처리 (주문완료 → 예약완료)

ㆍ고객이 예약을 완료하면 시스템상에서 다음과 같은 흐름으로 데이터가 처리됩니다.
①주문 생성: 고마스터 > 폐쇄몰(구분) > '주문완료'(주문상태) 상태로 예약건 자동 생성
②시스템 이관: 윙스(Wings) PMS로 데이터 생성 (시스템 이관 또는 필요 시 수기 등록)
③예약 확정: 객실마감여부 확인 후 고마스터 '확정' 처리

## 이관 및 고마스터 확정

### 1. 리저브클럽 예약 이관 (고마스터 → 윙스)

1. 데이터 추출 : 고마스터 → [윙스] 버튼 클릭 → 구분(리저브) 및 시설 선택 후 파일 다운로드
2. 양식 매칭 : 윙스 → [Reservation Excel Upload] 접속 → 양식(IR12_0100) 다운로드 → 고마스터 데이터 복사/붙여넣기 → 파일 저장
3. 업로드 : 윙스 내 [+] 버튼 클릭 → 저장한 파일(IR12_0100) 첨부 → [예약] 클릭 → 이관 완료 시 창 닫기

### 2. 객실 현황 확인 (오버부킹)

윙스 → 사용 가능 객실 메뉴에서 실시간 잔여 객실 확인

오버부킹 발생 시 즉시 고객 안내 진행, 윙스와 고마스터 모두 취소 처리

| 시설 | 객실 코드 |
|------|------|
| 오션투유 | SFS(포레스트), SOS(사이드), POS(파노라마) |
| 아이파크 | SPD(더블), DOD(온돌), DFD+PFT(더블트윈), OCS+OCF(스위트), PDB/PDT/PSU(펫룸), DFT(트윈) |
| 마리비스타 | SDB(더블)+RDB(펫더블)+VRD(오션더블), SFT(패밀리트윈)+RFT(펫패밀리트윈)+VFT(오션패밀리트윈), STW(트윈) |

### 3. 고마스터 최종 확정 (윙스 → 고마스터)

1. 데이터 추출 : 윙스 → 전체 고객 목록 → 예약일자(이관일) 및 거래처(리저브클럽) 설정 후 조회 → 엑셀 다운로드
2. 최종 등록 : 고마스터 → [확정] 버튼 클릭 → 구분(리저브) 및 시설 선택 → 파일 첨부 → [등록]

> ※ 다운로드한 엑셀 형식을 반드시 **Excel 97-2003 통합 문서(.xls)** 로 변환하여 저장

## 예약 취소

### 취소 기본 규정

| 구분 | 내용 |
|------|------|
| 취소 기한 | 입실일 기준 2일 전까지 가능 |
| 고객 직접 취소 | 홈페이지 → 리저브클럽 → 마이페이지 → 예약현황 → 예약내역 상세 → [예약취소] 버튼 |

### 예약 상태별 처리 절차

**주문완료 상태일 때 (입실 전 확정 대기)**

| 취소 주체 | 처리 절차 |
|------|------|
| 고객 직접 취소 | 시스템상 주문취소 상태로 자동 변경 (숙박권 자동 복구 + 결제 자동 취소) |
| 업무 담당자 취소 | ① 결제건 있을 경우 결제 수기취소 → ② 고마스터 취소 처리 (이 시점에 숙박권 자동 복구) |

**예약완료 상태일 때 (윙스 이관 및 확정 후)**

| 취소 주체 | 처리 절차 |
|------|------|
| 고객 직접 취소 | ① 취소대기 상태로 접수 (결제 자동 취소, 숙박권 복구X) → ② 윙스 예약 취소 → ③ 고마스터 취소 처리 (이 시점에 숙박권 자동 복구) |
| 업무 담당자 취소 | ① 윙스 예약 취소 → ② 결제 수기취소 → ③ 고마스터 취소 처리 (이 시점에 숙박권 자동 복구) |

> ※ 숙박권 복구 시점 : 고마스터 시스템 내 취소 처리가 완료될 때
> ※ 결제 취소 시점 : 고객이 취소 접수를 완료하는 즉시
> ※ 실제 환불 기간 : 취소 승인 후 영업일 기준 3~5일 소요


## 상품 취소 및 환불

| 구분 | 내용 |
|------|------|
| 취소 가능 기간 | 상품 구입 후 14일 이내 |
| 취소 불가 조건 | 14일 이내라도 온라인 예약을 통해 이용이 시작된 경우 환불 불가 |
| 고객 취소 경로 | 마이페이지 → 정보수정 → [상품 취소] |

**업무 담당자 취소 시**

1. 리저브클럽 구매리스트 취소
2. 결제 취소
3. 리저브클럽 관리자페이지 → 사용자 관리 → 회원 → 고객정보 검색 → 관리 → 정보수정 → 승인 → 이용중을 결제취소로 변경 → 수정


## 예약 후 부대시설 처리

### 1. 현장 사용하기(가장 간편한 방법)

고객이 현장에서 직접 모바일 화면을 제시하여 이용하는 방식
경로 : 홈페이지 > 리저브클럽 > 마이페이지 > 이용권 현황 > [부대시설 현장 사용하기]
방법 : 해당 화면을 프론트에 제출하여 현장에서 즉시 사용 처리(처리 시 관리자에서 이용권 자동 사용처리)

> ※ 당일 현장 상황에 따라 이용이 제한될 수 있음
> ※ 구매자가 아닌 다른 고객(지인 등)이 갈 경우 화면제시 불가한 경우가 있음

### 2. 취소 후 재예약 (권장 방법)

기존 예약을 취소하고 변경된 부대시설을 포함하여 다시 예약하는 방식
선행 조건: 기존 예약의 취소 가능 여부 및 재예약 시점의 객실 잔여 수량 확인 필수

> ※ 고객 마이페이지 내 예약 현황에서 변경된 부대시설 상세 내역 확인 가능
> ※ 추후 예약 취소 시에도 부대시설 이용권이 시스템상으로 정상 복구 및 반영됨

### 3. 수기 반영 (운영자 직접 수정)

시스템상 재예약이 어렵거나 긴급한 경우 담당자가 직접 각 시스템에 반영하는 방식
처리 절차

> ※ 당일 현장 상황에 따라 이용이 제한될 수 있음
> ※ 구매자가 아닌 다른 고객(지인 등)이 갈 경우 화면제시 불가한 경우가 있음
