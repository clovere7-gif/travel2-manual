# 예약실 취소 및 변경 기준

## 1. 기준

### 내용 파악
문의 인입 시 내용 파악 후 **일반취소인지 예외취소인지**, 강성취소 또는 컴플레인인지 파악 필요.

### 예약 취소 가능 여부 (고객이 예약한 날짜 기준 / 00시 기준)

| 구분 | 가능 여부 |
|------|-----------|
| 당일 예약 진행 | 가능 |
| 전일 예약 진행 | 가능 |
| 예약한 지 2일 이상 경과 | 불가 |

> 당일 예약건의 경우 **업무 시간 16시 기준**, 이후 취소 및 변경 불가
> 시간 내 회신 없을 경우 노쇼 처리 및 환불 불가
> 무취동까지만 메일 인입 시 야간에 인계

### 해외 채널 기준
체크인 날짜 기준 **5일 전부터 취소 불가**로 인지.

### 취소규정 이외 취소 요청
증빙서류 수취 필요 (예: 직계 장례, 교통사고 등)

> ⚠️ 증빙서류 수취하기로 한 경우 **윙스 선취소 필수**

## 2. 고객 안내 메뉴얼

### 고객 인폼 (유선 / 메일)

**예약사이트 전산과 호텔 전산은 다르다**

- 예약 취소는 바로 안 되고, 객실 배정만 먼저 취소한다고 안내
- 서류 받기로 한 순간 철회 불가
- 손님이 다시 재입실 여부, 시간이 걸려도 되는지, 직원 상황 인지 가능 여부 확인
- 호텔 전산 취소는 가능하나, 고객이 결제한 채널 전산은 취소 안 됨 → 환불은 채널로 문의 안내
- 예약실 직접 컨택 시(예: 아고다 채팅메시지) 예약사이트를 통해 먼저 문의하도록 안내 (현장톡 제외)

> 교육 시 자세한 설명 필요

## 3. 증빙서류 확인사항

증빙서류는 **예약자 본인과 관련되어 있는지 체크 필수**

| 구분 | 처리 기준 |
|------|-----------|
| 신분증 + 사망진단서 | 서류 확인 후 처리 |
| 공무원 / 군인 | 무조건 처리 아님, 캐물은 후 판별 — 공무원증 / 직업군인증 / 현역신분증 수취 |
| 거짓말 의심 시 | 신분증이라도 수취 후 판별 |
