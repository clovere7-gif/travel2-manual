# 마리비스타

## 기본정보

| 구분 | 내용 |
|------|------|
| 부대시설 | 조식당, 바베큐장, 세미나실, 룸서비스, 카페(테이크아웃), 키즈존 |
| 펫룸 | 더블, 트윈, 패밀리트윈, 온돌 가능 (코드 없음) |
| 주차 | 한정 150대 정도 |
| 특이사항 | 도보특가 상품 있음 / NPK 서비스코드 안들어가있을시(미연동) 기입필요 |
| 비고 | 취사불가 / 스위트 취사가능 |


## 객실코드

| 코드 | 상품명 |
|------|------|
| SDB | 시티뷰 스탠다드 더블 |
| STW | 시티뷰 스탠다드 트윈 |
| SFT | 시티뷰 스탠다드 패밀리 트윈 |
| FOD | 시티뷰 스탠다드 패밀리 온돌 |
| RDB | 시티뷰 슈페리어 더블 |
| RTW | 오션시티뷰 슈페리어 트윈 |
| RFT | 시티뷰 슈페리어 패밀리 트윈 |
| RFO | 시티뷰 슈페리어 패밀리 온돌 |
| VRD | 오션레이크뷰 슈페리어 더블 |
| VFT | 오션레이크뷰 슈페리어 패밀리 트윈 |
| CST | 오션레이크뷰 코너 스위트 |
| DFS | 시티뷰 디럭스 패밀리 스위트 |
| VFS | 오션레이크뷰 디럭스 패밀리 스위트 |


## 업그레이드 순서

시티뷰 → 오션뷰

> ※ 객실별 구조가 달라 호환되지 않아 뷰만 매칭해서 업그레이드
> ※ 트윈 → 패밀리트윈 업그레이드 절대금지
