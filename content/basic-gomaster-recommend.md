# 고마스터 예약 / 관리자추천

## 1. 예약리스트 메뉴 진입

고마스터 접속 후 상단 메뉴 [예약관리] > **[전체] 예약리스트** 클릭합니다.

![예약관리 메뉴에서 전체 예약리스트 선택](images/gomastere/basic-gomaster-recommend-01.png)

## 2. 전체 예약목록 확인

[전체] 예약리스트 > **관리자추천** 클릭합니다.

![전체 예약리스트 화면](images/gomastere/basic-gomaster-recommend-02.png)

## 3. 빨간칸 모두 입력후 (하단참조) > 관리자추천 클릭

![전체 예약리스트 화면](images/gomastere/basic-gomaster-recommend-03.png)

3-2 예약자명, 전화번호 입력 
전화번호없거나 입력필요없을시 [X] 클릭 010-0000-0000 으로
3-3 투숙일 선택 / 당일일경우 [당일]버튼클릭 오늘날짜로 세팅
3-4 업체명 시설버튼 클릭
3-5 룸타입 ⓞ 선택
3-6 판매가, 입금가 입력 / 띄어쓰기, 빈칸, ","(쉼표) 모두입력가능
3-7 관리자추천

## 3-1.구분선택 클릭후 / 대분류 > 채널선택 > 대매사 주문번호 입력

구분선택 클릭후 / 대분류 > 채널선택 > 대매사 주문번호 입력
