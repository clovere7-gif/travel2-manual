# 고마스터 예약 / 관리자추천

## 1. 예약리스트 메뉴 진입

고마스터 접속 후 상단 메뉴 [예약관리] > **[전체] 예약리스트** 클릭합니다.

![예약관리 메뉴에서 전체 예약리스트 선택](/images/gomastere/basic-gomaster-recommend-01.png)

## 2. 전체 예약목록 확인

[전체] 예약리스트 > **관리자추천** 클릭합니다.

![전체 예약리스트 화면](/images/basic-gomaster-recommend-02.png)

## 3. 빨간칸 모두 입력후 (하단참조) > 관리자추천 클릭

![전체 예약리스트 화면](/images/basic-gomaster-recommend-03.png)

## 3-1.구분선택 클릭후 / 대분류 > 채널선택 > 대매사 주문번호 입력

![예약 상세내용 화면](/images/basic-gomaster-recommend-04.png)
