# 스마트스토어 구매처리

**판매 페이지** — https://smartstore.naver.com/brmall

> ※ 취소 규정은 자사몰 판매 리저브클럽과 동일
> ※ 구매인증번호는 영업일 3일 이내에 발송

**관리자 페이지** — https://sell.smartstore.naver.com/#/home/about

## 구매 처리 순서

1. 고객 구매 → 판매관리 → 발주 확인/발송관리 → 신규주문(발주 후)으로 인입

> ※ 배송없음 상품이기 때문에 신규주문(발주 전)으로 들어오지 않음

![스마트스토어 신규주문 화면](images/reserve/reserve-07.PNG)

2. 검색 후 회원 정보를 리저브 관리자페이지, 스마트스토어(리저브) 엑셀 시트에 등록

- 결제 아이디 — '(유튜버)기프트팩_제공' 시트 데이터와 대조하여 중복되지 않은 미사용 번호 확인 후 순차 할당
- 인증번호 — 리저브 관리자 페이지에서 회원 등록하여 인증번호 생성 후 입력
- 취소 가능일 — 시트 내 수식이 이미 설정되어 있으므로 별도 설정 및 수정 불필요

![스마트스토어 주문 검색 화면](images/reserve/reserve-08.PNG)

**② 리저브클럽 관리자페이지 고객 등록**

- 등록 경로 — 관리자 페이지 → 사용자 관리 → 회원 → 등록
- 정보 등록 — 스마트스토어 엑셀 시트의 결제 아이디와 이메일 주소를 동일하게 입력
- 유효기간 설정 — 등록 일자 + 1년/2년 - 1일 기준으로 설정

![스마트스토어 엑셀 시트 등록](images/reserve/reserve-09.PNG)

**③ 구매, 안내 카톡 발송**

발송 경로 — 관리자 페이지 → 사용자 관리 → 회원 → SMS → 구매/안내 클릭

![리저브 관리자페이지 회원 등록](images/reserve/reserve-10.PNG)

3. 등록 후 발송처리

![구매/안내 카톡 발송](images/reserve/reserve-11.PNG)

4. 발송 처리 시 판매관리 → 배송현황 관리에서 확인 가능

![발송처리 화면](images/reserve/reserve-12.PNG)

5. 고객 구매확정 또는 자동 구매확정 시 판매관리 → 구매확정 내역에서 확인 가능

![구매확정 내역 화면](images/reserve/reserve-13.PNG)
