# 예약

## 1. 메일 확인

담당메일 로그인 후 개인메일함 **네이버**에서 예약 확인

![메일함 네이버 탭](images/ota/naver/ota-portal-naver-reserve-01.PNG)

## 2. 예약진행 (전체 미연동)

![알림메일 자세히 보기](images/ota/naver/ota-portal-naver-reserve-02.PNG)

예약 알림메일 클릭 후 하단 **자세히 보기** 클릭

ⓐ 네이버 아이디 로그인 또는 ⓑ 로그인되어 있을 경우 예약건 새창팝업

![ⓐ 로그인화면 / ⓑ 예약건 새창팝업](images/ota/naver/ota-portal-naver-reserve-03.PNG)

## 2-1. 예약입력

![신규예약 입력화면](images/ota/naver/ota-portal-naver-reserve-04.PNG)

> ※ ⑤ 건드리지 말 것

## 2-2. 예약입력

![고마스터 입력 완료화면](images/ota/naver/ota-portal-naver-reserve-05.PNG)

1. 고마스터 → 관리자추천 → 네이버 클릭 후 예약번호가 아닌 ③NPay 주문번호 입력, 기본인적사항 기재
2. 날짜, 시설, 객실 입력
3. 결제금액을 고마스터 판매가에 입력 후 3.19%(네이버수수료) 클릭하면 입금가 자동산출
4. 고마스터 입력완료
5. 윙스예약은 기본-3 진행
6. 예약완료

# 네이버 예약 - 오션투유리조트 다연예약

## 1. 메일 확인

로그인 후 개인메일함 **네이버**에서 예약 확인 — 오션투유리조트 다연 제목 확인

![다연 알림메일 바우처](images/ota/naver/ota-portal-naver-reserve-06.PNG)

## 2. 예약진행

1. 고마스터 → 관리자추천 → 네이버 클릭 후 예약번호 입력, 기본인적사항 기재
2. 이름 옆에 현장결제, 기타요청사항 혹은 메모에 이용인원 기재
3. 날짜, 오션투유리조트, 조식 디럭스 선택
4. 결제금액을 판매가 입력가 동일하게 입력
5. 고마스터 입력완료
6. 예약완료

![고마스터 입력 완료화면](images/ota/naver/ota-portal-naver-reserve-07.PNG)
