# 온다 예약

## 업무범위

1. 메일 확인
2. 예약진행 (전체 미연동)
3. 객실마감, 요금수정
4. 완료예약확인

---

## 예약

### 1. 메일 확인

담당메일 로그인 후 개인메일함 **소셜**에서 예약 확인

![하이웍스 소셜 메일함](images/ota/onda/ota-social-onda-reserve-01.PNG)

---

### 2. 예약진행 (전체 미연동)

예약 알림메일 확인 후 관리자 사이트 접속

![온다 알림메일 바우처](images/ota/onda/ota-social-onda-reserve-02.PNG)

**알림메일(바우처) 확인 항목**

1. 예약번호
2. 고객정보
3. 객실 및 상품정보
4. 날짜
5. 입금가 / 수수료
6. 판매가
7. 예약확정 후 호텔 예약번호 입력 (오션투유 → 성함+입실(OOO) 혹은 윙스 번호)

![관리자 사이트 예약목록](images/ota/onda/ota-social-onda-reserve-03.PNG)

> 알림메일 및 관리자 사이트 확인 후 예약완료 처리
> 7번 호텔 예약번호 입력 누락 주의 — 미입력건 발견 시 즉시 입력

---

## 객실마감, 요금수정

### 객실마감

![요금/재고관리 마감 설정](images/ota/onda/ota-social-onda-reserve-04.PNG)

**관리자페이지 순서**

1. 요금/재고관리 선택
2. 판매상태 클릭
3. 판매 마감 버튼 활성화 — 해당 버튼으로 마감/판매 전환

---

### 요금수정

![요금/재고관리 요금수정](images/ota/onda/ota-social-onda-reserve-05.PNG)

1. 요금/재고관리 선택
2. 재고수량 변경 필요 시 변경
3. 판매가 기준으로 변경
