# 예약

## 1. 메일확인

메일 로그인 후 개인메일함 **받은메일함**에서 예약 확인

![하이웍스 여행사 메일함](images/ota/tripotaz/ota-portal-tripotaz-reserve-01.PNG)

## 2. 예약진행 (연동)

예약 알림메일 확인 후 알림메일에서 예약

![트립토파즈 알림메일 바우처](images/ota/tripotaz/ota-portal-tripotaz-reserve-02.PNG)

**알림메일(바우처) 확인 항목**

1. 예약번호
2. 고객정보
3. 날짜
4. 판매가 / 입금가
5. 객실 및 상품정보
6. 윙스 번호

> 예약완료 후 관리자 사이트에서 예약확정 처리

**예약확정 처리 순서**

![관리자 사이트 예약확정 처리](images/ota/tripotaz/ota-portal-tripotaz-reserve-03.PNG)

1. 시설 확인
2. 미확인 예약 클릭

![관리자 사이트 예약확정 처리](images/ota/tripotaz/ota-portal-tripotaz-reserve-04.PNG)

1. 확정해야 하는 예약 선택
2. 상세보기 클릭
3. 예약번호란 — 오션투유의 경우 성함+입실(OOO), 연동의 경우 윙스번호 기재되어 있음 → 예약확인 클릭
