# 전화 응대

## 1. 메모 작성 방법

메모 = 고마스터 메모를 뜻함

| 구분 | 내용 |
|------|------|
| 문의들어온곳 | 현장카톡 > / 오션팀룸, 고객유선 등 |
| 내용 | 핵심만 정리 |
| 처리상황 | 핵심만 정리 |
| 고객반응 | 고객수긍 / 고객불수긍 / 재통화요청 / CP 등 |
| 특이사항 | 상급자 요청 / 명일통화예정 등 |

**작성 예시**

현장카톡 > 010-XXXX-XXXX 리저브 예약 취소 관련 통화요청 / 통화완료, 예약취소 직접처리 안내 / 고객수긍 / 없음

## 2. 현장 전화 응대 - 기본

### 케이스 A. 처리 가능한 문의
전화 수신 → 인사 및 문의사항 청취 → 답변 → 감사 인사 종료 → 메모 작성 → **완료**

### 케이스 B. 처리 불가능한 문의
전화 수신 → 인사 및 문의사항 청취 → "확인 후 연락드리겠습니다" 종료 → 메모 작성 → 상급자/담당자 문의 → 메모 작성 → 고객 재안내 → **완료**

### 케이스 C. 고객이 먼저 인사/수고 멘트를 할 경우
전화 수신 → 고객 인사/수고 → 같이 인사/감사 → 문의사항 청취 → 케이스 A 또는 B로 이어짐

### 케이스 D. 문의사항을 못 알아들은 경우
전화 수신 → 인사 → 문의사항 재질문 → 케이스 A 또는 B로 이어짐

> 모든 케이스에서 통화 종료 후 반드시 메모 작성

---

## 3. 리저브 · 홈쇼핑 전화 응대

전화 수신 전 → 전화번호로 리저브/홈쇼핑 관련 정보 미리 확인 및 문의사항 시뮬레이션

### 케이스 A. 예약 내역 없음
구매자/전화번호/예약내역 검색 → 예약 내역 없음 확인 → "직접 예약해주세요" 안내

### 케이스 B. 취소 요청
예약 내역 확인 → 취소 요청 → "직접 취소해주세요" 안내
- 방법을 모르는 경우 → "직접 하셔야 합니다" 재안내

### 케이스 C. 변경 요청
예약 내역 확인 → 변경 요청 → 객실 가능 여부 확인
- 가능 시 → "직접 변경해주세요" 안내
- 마감 시 → 불가 답변
  - 그래도 말을 안 들을 경우 → 불가 재답변
  - 계속 말을 안 들을 경우 → "확인 후 연락드리겠습니다"

---

## 4. 일반예약 - 자사

전화 수신 → 내용 파악 → 처리 및 취소 예약실에서 직접 처리

> 취소, 결제취소, 윙스 취소 등 놓치지 않도록 주의

---

## 5. 일반예약 - OTA

### 케이스 A. 처리 가능
내용 파악 → 메모 작성 → 처리 후 현장 공유 → 메모 작성 → **완료**

### 케이스 B. 처리 불가능
내용 파악 → 메모 작성 → 팀룸 공유 → **완료**
