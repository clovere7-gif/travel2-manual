# 결제취소

## 기초. 부가세 계산기 사용법

![이미지](images/cancel/basic-cancel-linked-26.PNG)

| 구분 | 내용 |
|------|------|
| 합계 금액 | 취소할 금액 |
| 공급가액 | 과세 금액 |
| 부가세액 | 부가세 (비과세는 0원 처리) |

### ☆ 결제 취소 후 해야 하는 것

1. 결제 취소 사유 및 처리 내용 고마스터 메모 기재 **← 제일 중요**
2. 객실 전체 취소일 경우: 윙스 수기 취소 및 고마스터 취소
3. 조식 부분 취소일 경우: 취소 관련 내용 기재

> 

## 1. KCP 취소

링크: https://partner.kcp.co.kr/auth/login

- 오션투유 예약 결제 취소는 오션투유 전용 계정, 이 외 나머지 호텔 결제 취소는 공통 계정으로 로그인

### 케이스 A. 고객이 자사 홈페이지로 결제하여 예약한 경우

![이미지](images/cancel/basic-cancel-linked-21.PNG)

1. 고마스터에서 해당 고객 예약건 접속
2. KCP 로그인 후 신용카드 → 거래내역 조회 클릭

![이미지](images/cancel/basic-cancel-linked-22.PNG)

3. 승인 일자 = 고객이 예약한 일자
4. 주문번호에 고마스터 자체주문번호 입력 → 조회
5. 카드 번호 클릭

![이미지](images/cancel/basic-cancel-linked-23.PNG)

6. 상태변경 → 취소요청 버튼 클릭

![이미지](images/cancel/basic-cancel-linked-24.PNG)

7. 전액 취소인 경우: 취소 사유 선택 → 확인 누르면 취소 처리됨

### 케이스 B. 부분 취소인 경우 (예: 조식 부분 취소)

![이미지](images/cancel/basic-cancel-linked-25.PNG)

취소할 금액 = 조식 금액
예를 들어 , 고객님께서 조식패키지 객실 예약 > 조식만 부분 취소 요청하시는 경우
취소할 금액 = 조식 금액 , 과세 및 부가세 계산은 부가세 계산기를 이용
취소해줘야 하는 조식 금액 24,900원이라고 가정
상기 대로 부가세 계산기 사용

>

## 2. 이니시스 취소

링크: https://iniweb.inicis.com/nxaview/index.jsp

![이미지](images/cancel/basic-cancel-linked-27.PNG)

> 이니시스 취소의 경우, 고마스터 예약건 PG주문번호 앞에 "INIPAY" 기재되어 있음

![이미지](images/cancel/basic-cancel-linked-28.PNG)

1. 이니시스 전용 계정으로 로그인
2. 2차 인증 로그인 → 메일 인증

![이미지](images/cancel/basic-cancel-linked-29.PNG)

3. 로그인 후 서비스 목록 → 신용카드 클릭

![이미지](images/cancel/basic-cancel-linked-30.PNG)

4. 거래일자 = 고객 예약한 일자, TID = 고마스터 PG주문번호 입력 후 조회

![이미지](images/cancel/basic-cancel-linked-31.PNG)

5. ""TID"" 클릭 후 결제 취소
6. 취소 되었는지 재조회해서 취소 확인까지 하기

>

## 3. 카카오페이 취소

링크: https://accounts.kakao.com/login/

![이미지](images/cancel/basic-cancel-linked-32.PNG)

1. 예약 및 결제 취소 사유 메모에 기재
2. 카카오페이 전용 계정으로 로그인
3. 로그인 인증: 카카오톡 1차 인증 → 문자로 2차 인증번호 받아서 로그인 (메일도 가능하나 잘 안됨)

![이미지](images/cancel/basic-cancel-linked-33.PNG)

4. 카카오페이 로그인 후 TID에 고마스터 예약건 PG주문번호 입력 후 조회
5. 결제 건 클릭 후 취소

![이미지](images/cancel/basic-cancel-linked-34.PNG)

6. 취소 되었는지 검색 다시 눌러서 조회 확인

>

## 4. 네이버페이 취소

링크: https://admin.pay.naver.com/z/payments/list

![이미지](images/cancel/basic-cancel-linked-35.PNG)

1. 예약 및 결제 취소 사유 메모에 기재
2. 네이버페이 전용 계정으로 로그인
3. 2차 로그인 인증: OTP 인증 (담당자 휴대폰으로 확인)

![이미지](images/cancel/basic-cancel-linked-36.PNG)

4. 네이버페이 로그인 후 고마스터 예약건의 PG주문번호 입력 후 조회
5. 클릭 후 취소 처리
