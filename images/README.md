# 이미지 폴더 사용법

## 파일명 규칙

다음 형식으로 저장하는 것을 권장합니다.

```
채널명-기능-순번.png
```

예시:
- `야놀자-예약하기-01.png`
- `야놀자-예약하기-02.png`
- `윙스-객실수량확인-01.png`

## 주의사항

1. 띄어쓰기 대신 하이픈(-)을 사용하세요. (띄어쓰기가 있어도 동작은 하지만, 문제가 생기기 쉽습니다)
2. 대문자/소문자를 섞어 쓰지 마세요. (`Image.png`와 `image.png`는 다른 파일로 인식됩니다)
3. 스크린샷은 일반 해상도(웹/앱 화면 그대로)면 충분합니다. 4K 등 고해상도로 저장하지 마세요.

## md 파일에서 사용하는 법

content 폴더의 .md 파일에서 아래처럼 작성하면 이미지가 표시됩니다.

```
![화면 설명](../images/야놀자-예약하기-01.png)
```
