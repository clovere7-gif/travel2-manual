# 로컬에서 미리보기 하는 법

이 사이트(index.html)는 menu.json과 content 폴더의 .md 파일을 "불러오는" 방식으로 작동합니다.
그래서 index.html을 그냥 더블클릭해서 열면 보안 정책 때문에 내용이 비어 보입니다.

GitHub에 올리기 전에 미리 확인하고 싶다면, 아래 두 방법 중 하나를 사용하세요.

## 방법 1. Python으로 임시 서버 띄우기 (가장 간단)

1. 이 폴더(manual)에서 명령 프롬프트(또는 터미널)를 엽니다.
2. 아래 명령어를 입력합니다.

```
python -m http.server
```

3. 브라우저에서 다음 주소로 접속합니다.

```
http://localhost:8000
```

4. 종료할 때는 명령 프롬프트 창에서 Ctrl + C

## 방법 2. VS Code의 Live Server 확장 사용

1. VS Code 설치
2. 확장(Extensions) 탭에서 "Live Server" 검색 후 설치
3. index.html 파일을 우클릭 → "Open with Live Server"
4. 자동으로 브라우저가 열리며 미리보기 표시

## 확인이 끝났다면

수정한 파일(.md, menu.json, 이미지)을 GitHub 저장소에 업로드하면
실제 서비스 주소(GitHub Pages URL)에도 1~2분 내로 반영됩니다.
